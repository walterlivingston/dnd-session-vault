## Setup <a name = "setup"></a>

**Step 1: Setup Obsidian**
- Download Obsidian for your platform from this [link](https://obsidian.md/download)

**Step 2: Clone the Repo**
- Clone this repository to folder of your choice using the following command in Powershell or Terminal
- (Optional) If you don't have git setup on your system, follow [this guide](https://docs.github.com/en/get-started/getting-started-with-git/set-up-git)
```shell
git clone https://github.com/walterlivingston/dnd-session-vault.git
```

**Step 3: Open in Obsidian**
- Open the cloned folder in Obsidian as a Vault
- Make sure to trust the creator so you get all the fun plugins
> These plugins don't work on the site, but they make editing more fun

**Step 4: Publishing changes**
- Ensure you are in the root directory of the repository and run the following commands in Powershell or Terminal
```shell
git add .
git commit -m "PUT A COMMIT MESSAGE EXPLAINING CHANGES HERE"
git push
```

## Common Markdown Techniques <a name = "markdown-techniques"></a>
**Italicizing**
- Use single asterisks( * * ) to italicize the text within

**Bolding**
- Use double asterisks( ** ** ) to bold the text within

**Linking**
- Use double square brackets( [[ ]] ) to link a word to another file

**Images**
- Use an exclamation and double square brackets ( ![[ ]] ) to embed an image
